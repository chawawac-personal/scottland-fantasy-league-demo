# Scottland Fantasy League

## What it does
Fantasy football platform for Zimbabwean club football. Players build squads, compete in leagues, track live match stats, and earn fantasy points. Includes a manager role for league administration and a community feed.

## User Roles

| Role | Access |
|------|--------|
| `admin` | Full platform control — all users, leagues, match data, feature flags |
| `manager` | League management — create and moderate leagues, manage fixtures |
| `user` | Standard player — build squad, join leagues, view standings and live scores |
