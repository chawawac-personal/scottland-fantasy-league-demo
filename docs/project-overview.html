SCOTTLAND FC FANTASY LEAGUE

Project Overview — Prepared by OMNI Global · 2026

CONFIDENTIAL

The Scottland FC Fantasy League is a bespoke digital fan engagement platform that bridges passionate supporters and real-time match action. Fans draft real club players, earn live fantasy points based on actual match statistics, and compete on weekly leaderboards — transforming casual viewers into highly active team managers and driving year-round loyalty to the Scottland FC community.

Core Features
⚽

Fantasy Team Builder

Pick 11 starters from SFC's 18-player squad within a budget. Set captain (2×) and vice-captain (1.5×) for bonus points.

📡

Live Matchday Centre

Real-time score updates, event feed (goals, cards, assists) and a live personal points tracker updating as events are logged.

🏆

Private & Public Leagues

Fans create private leagues with invite codes or join public competitions. Prize descriptions configurable per league.

📊

Player Market

Buy and sell players within budget. Search, filter by position, and sort by stats — with live ownership percentages.

🎖

Gamification

XP points, level titles (Rookie → Legend), achievement badges, and a weekly digest keep fans engaged year-round.

📣

Fan Community

Fan polls, matchday chat with emoji reactions, and a discussion board foster an active supporter community.

Business Opportunity
  • Sponsor visibility via branded mini-leagues and prize giveaways
  • Fan data for targeted club communications and merchandise
  • Year-round engagement — not limited to match days
  • Matchday attendance incentivised through league rewards
  • Official partner placement within polls and notifications
Technology Stack
  • Frontend: Next.js 14 (React, TypeScript)
  • Backend / DB: Supabase (PostgreSQL + Row-Level Security)
  • Realtime: Supabase Realtime channels (WebSockets)
  • Hosting: Vercel (global edge deployment)
  • Auth: Supabase Auth (email + password)
  • Styling: Tailwind CSS + Framer Motion
