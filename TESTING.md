## Engines

### Scoring Engine
- Triggered automatically on `INSERT`/`UPDATE` to `player_match_stats`
- `calculate_player_match_points(position, goals, assists, yellows, reds, clean_sheet, minutes)` — point formula
- `trg_calc_stat_points` — BEFORE trigger, auto-fills `fantasy_points` on each stat row
- `trg_rollup_player_totals` — AFTER trigger, updates `players.total_points`, `goals`, `assists`, etc.
- `recalculate_matchday_team_points(matchday, season)` — run after all stats entered; updates all user fantasy_points, league ranks, then calls achievements engine

### Achievements + XP Engine
- `award_achievements(user_id)` — evaluates all badge conditions for one user
- `award_all_achievements()` — runs for every user; called automatically at end of `recalculate_matchday_team_points`
- `grant_xp(user_id, xp)` — adds XP, levels up if threshold crossed, sends level-up notification
- Badges: `top_scorer` (top 100), `top_manager` (top 10), `trophy_md_winner`, `trophy_fan_favourite`, `transfer_master`, `die_hard`, `unbeaten`

### League Ranking Engine
- Runs inside `recalculate_matchday_team_points` — re-ranks all `league_members` by points within each league
- `weekly_points` and `points` on `league_members` kept in sync with `fantasy_teams`

### Poll Votes Engine
- `cast_poll_vote(poll_id, option)` — SECURITY DEFINER, prevents double-voting, updates `polls.votes` JSONB atomically

### Auth / Profile Trigger
- `handle_new_user()` on `auth.users INSERT` — creates `profiles` row automatically on registration
- Reads `username`, `full_name`, `phone` from `raw_user_meta_data`

### Login Resolver
- `resolve_login_identifier(identifier)` — SECURITY DEFINER; maps username or phone → email for the login page
- Supports email, username, or phone number at the login screen

## Key Test Flows

### Registration
1. Go to `/register`
2. Fill in Full Name, Username, and either Email OR Phone (one is enough)
3. Submit → lands on `/privacy?from=onboarding`
4. Accept privacy policy → `/onboarding`
5. Complete onboarding → `/dashboard`
- Email confirmation is **disabled** — registration is instant

### Login
Accepts any of: email address, phone number (`+263 77…`), or `@username`

### Admin — Enter Match Stats
1. Sign in as SFCAdmin
2. Go to **Manager Panel** (sidebar)
3. Click **Score** on a finished match
4. Enter goals, assists, minutes, cards per player
5. Click **Save & Recalculate** → scoring engine fires, all user points update instantly

### Admin — Send Notification
1. Admin Panel → Notifications tab
2. Fill title, body, type → Send to All
3. Users see it in the TopBar bell icon
