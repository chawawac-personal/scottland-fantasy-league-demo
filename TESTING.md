## Engines

### Scoring Engine
- Triggered automatically on `INSERT`/`UPDATE` to `player_match_stats`
- `calculate_player_match_points(position, goals, assists, yellows, reds, clean_sheet, minutes)` ŌĆö point formula
- `trg_calc_stat_points` ŌĆö BEFORE trigger, auto-fills `fantasy_points` on each stat row
- `trg_rollup_player_totals` ŌĆö AFTER trigger, updates `players.total_points`, `goals`, `assists`, etc.
- `recalculate_matchday_team_points(matchday, season)` ŌĆö run after all stats entered; updates all user fantasy_points, league ranks, then calls achievements engine

### Achievements + XP Engine
- `award_achievements(user_id)` ŌĆö evaluates all badge conditions for one user
- `award_all_achievements()` ŌĆö runs for every user; called automatically at end of `recalculate_matchday_team_points`
- `grant_xp(user_id, xp)` ŌĆö adds XP, levels up if threshold crossed, sends level-up notification
- Badges: `top_scorer` (top 100), `top_manager` (top 10), `trophy_md_winner`, `trophy_fan_favourite`, `transfer_master`, `die_hard`, `unbeaten`

### League Ranking Engine
- Runs inside `recalculate_matchday_team_points` ŌĆö re-ranks all `league_members` by points within each league
- `weekly_points` and `points` on `league_members` kept in sync with `fantasy_teams`

### Poll Votes Engine
- `cast_poll_vote(poll_id, option)` ŌĆö SECURITY DEFINER, prevents double-voting, updates `polls.votes` JSONB atomically

### Auth / Profile Trigger
- `handle_new_user()` on `auth.users INSERT` ŌĆö creates `profiles` row automatically on registration
- Reads `username`, `full_name`, `phone` from `raw_user_meta_data`

### Login Resolver
- `resolve_login_identifier(identifier)` ŌĆö SECURITY DEFINER; maps username or phone ŌåÆ email for the login page
- Supports email, username, or phone number at the login screen

## Key Test Flows

### Registration
1. Go to `/register`
2. Fill in Full Name, Username, and either Email OR Phone (one is enough)
3. Submit ŌåÆ lands on `/privacy?from=onboarding`
4. Accept privacy policy ŌåÆ `/onboarding`
5. Complete onboarding ŌåÆ `/dashboard`
- Email confirmation is **disabled** ŌĆö registration is instant

### Login
Accepts any of: email address, phone number (`+263 77ŌĆ”`), or `@username`

### Admin ŌĆö Enter Match Stats
1. Sign in as SFCAdmin
2. Go to **Manager Panel** (sidebar)
3. Click **Score** on a finished match
4. Enter goals, assists, minutes, cards per player
5. Click **Save & Recalculate** ŌåÆ scoring engine fires, all user points update instantly

### Admin ŌĆö Send Notification
1. Admin Panel ŌåÆ Notifications tab
2. Fill title, body, type ŌåÆ Send to All
